# Vortex — Premiers pas

Bienvenue !

## Démarrage rapide

1. **Lancer** `Vortex.exe` (mode `apply` par défaut)
2. **Voir les effets** : ouvre ton jeu ou ton appli lourde
3. **Vérifier la télémétrie** : `vortex.exe dash`

## Si quelque chose ne va pas

`vortex.exe restore` — remet tout à l'état antérieur (sauvegarde auto dans `C:\ProgramData\Vortex\`)

## À propos du projet

Vortex est un projet **ère simbiotique** : on ne *remplace pas* ton matériel, on lui parle dans sa langue. Les registres, les services, le kernel scheduler — Vortex configure Windows pour ton PC précis, pas pour une moyenne statistique.
